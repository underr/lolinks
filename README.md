# lolinks

### Install

`git clone https://github.com/underr/lolinks.git`

`npm install`

`cp config.js.example config.js`

`npm start`

### Administration

* Hover links to see their IDs

* Delete links using the address `/delete/STRING/ID` where **STRING** must be in your `config.js` file
